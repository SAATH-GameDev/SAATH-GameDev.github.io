But why bother?

  1. To learn developing games with like-minded people supported by mentors and other gamedevs.
  2. To learn any gamedev-related skill by 'doing' it without worrying about the rest of the game.
  3. To learn leading a team, managing a game project and releasing it on time.
  4. To improve communication, public speaking and remote work ethics.
  5. To develop games as a sustainable light-weight hobby.
  6. To add more in your games portfolio.
